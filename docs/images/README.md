# Documentation Images

This directory contains screenshots and images used in the project documentation.

## Organization

- `ui-*.png` - UI screenshots (interface, panels, controls)
- `setup-*.png` - Setup and installation screenshots (SSL warnings, permissions, etc.)
- `demo-*.png` - Demo screenshots showing functionality
- `architecture-*.png` - Architecture diagrams

## Adding New Images

1. Take screenshots at reasonable resolution (1920x1080 or similar)
2. Use descriptive filenames (e.g., `ui-main-interface.png`, `setup-ssl-warning.png`)
3. Optimize images before committing (use `pngquant` or similar)
4. Reference in markdown: `![Alt text](docs/images/filename.png)`
